# n8n Cricket Live Score Bot

## Project Overview

This workflow fetches live cricket scores using the Cricbuzz API and sends updates to Telegram automatically.

## Workflow

Schedule Trigger → Live Matches API → Match ID Extraction → Scorecard API → Score Extraction → Telegram Bot

## Technologies Used

* n8n
* RapidAPI (Cricbuzz Cricket API)
* Telegram Bot API
* JavaScript

## Features

* Fetch live cricket matches
* Extract batsman score details
* Send score updates to Telegram
* Can be scheduled to run automatically
